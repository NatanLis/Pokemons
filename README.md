
# Pokédex
### Pokedéx with Nuxt 3 + Typescript + Tailwind

## Commends
- Install dependencies
``` bash
yarn install
```
- Run application.
``` bash
 yarn dev
```

## Comments
Application hosted at <a href="http://localhost:3000/">http://localhost:3000/</a>.
